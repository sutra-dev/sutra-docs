title: SUTRA-V2 with Node.js SDK
---

This guide demonstrates how to use the **SUTRA-V2** model with the Node.js SDK to build scalable, multilingual AI applications. SUTRA-V2, developed by TWO AI, supports chat and instruction tasks via an OpenAI-compatible API.

## üîß Prerequisites

- Obtain your SUTRA API key from [https://www.two.ai/sutra/api](https://www.two.ai/sutra/api).
- Store the API key securely (e.g., as an environment variable).

## üì¶ Step 1: Install Dependencies

```bash
npm install openai
```

## üîê Step 2: Initialize SUTRA Client

```javascript
const { OpenAI } = require("openai");

const client = new OpenAI({
  apiKey: process.env.SUTRA_API_KEY || "YOUR_SUTRA_API_KEY",
  baseURL: "https://api.two.ai/v2",
});
```

## üí¨ Step 3: Basic Chat Completion

```javascript
async function main() {
  try {
    const response = await client.chat.completions.create({
      model: "sutra-v2",
      messages: [
        {
          role: "system",
          content: "You are a helpful AI that answers concisely.",
        },
        { role: "user", content: "Explain AI in 3 sentences in Hindi." },
      ],
      max_tokens: 1024,
      temperature: 0.7,
    });
    console.log(response.choices[0].message.content);
  } catch (error) {
    console.error("Error:", error.message);
  }
}

main();
```

## üõ† Troubleshooting

- **Invalid API Key**: Ensure `SUTRA_API_KEY` is set correctly in your environment (e.g., `.env`).
- **Model Not Found**: Use `sutra-v2`. SUTRA-V1 was deprecated on March 22, 2025.
- **Rate Limits**: Reduce request frequency or contact TWO AI at [https://www.two.ai/support](https://www.two.ai/support).
