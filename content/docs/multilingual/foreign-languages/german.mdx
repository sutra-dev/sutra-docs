Dieses Notebook zeigt, wie man **SUTRA-V2** auf **Deutsch** verwendet, um mehrsprachige KI-Anwendungen zu erstellen. Entwickelt von TWO AI, unterstützt SUTRA-V2 über 50 Sprachen, einschließlich Deutsch, für Chat- und Anweisungsaufgaben über eine OpenAI-kompatible API.

## 🔧 Voraussetzungen

- Besorgen Sie sich Ihren SUTRA-API-Schlüssel von [https://developer.two.ai](https://developer.two.ai).
- Speichern Sie den API-Schlüssel sicher (z. B. in Google Colab Secrets oder als Umgebungsvariable).

## 📦 Schritt 1: Abhängigkeiten installieren

```bash
!pip install -qU openai
```

## 🔐 Schritt 2: SUTRA-Client initialisieren

```python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_SUTRA_API_KEY",
    base_url="https://api.two.ai/v2"
)
```

## 💬 Schritt 3: Grundlegende Chat-Vervollständigung

```python
response = client.chat.completions.create(
    model="sutra-v2",
    messages=[
        {"role": "system", "content": "Sie sind eine hilfreiche KI, die kurz und bündig antwortet."},
        {"role": "user", "content": "Was ist künstliche Intelligenz? Erklären Sie in drei Sätzen."}
    ],
    max_tokens=1024,
    temperature=0.7
)
print(response.choices[0].message.content)
```

## 🛠 Fehlerbehebung

- **Ungültiger API-Schlüssel**: Überprüfen Sie Ihren Schlüssel unter [https://developer.two.ai](https://developer.two.ai).
- **Modell nicht gefunden**: Verwenden Sie `sutra-v2`. SUTRA-V1 wurde am 22. März 2025 eingestellt.
- **Ratenbeschränkungen**: Reduzieren Sie die Anfragehäufigkeit oder kontaktieren Sie TWO AI unter [https://www.two.ai/support](https://www.two.ai/support).
