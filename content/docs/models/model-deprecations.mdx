- Visit [docs.two.ai](https://docs.two.ai) to confirm your model version and review API usage.

## üîê Step 2: Obtain or Verify Your API Key

To access newer models, ensure you have a valid SUTRA API key:

- Get your API key from [https://www.two.ai/sutra/api](https://www.two.ai/sutra/api).
- Set your API key as an environment variable:

```bash
export SUTRA_API_KEY="your-api-key-here"
```

Then in your Python code:

```python
import os
SUTRA_API_KEY = os.getenv("SUTRA_API_KEY")
```

## üîÑ Step 3: Transition to SUTRA-V2

For SUTRA-V1 users, update your code to use SUTRA-V2:

- **Update Model Name**: Replace `sutra-v1` with `sutra-v2` in your API calls.
- **Initialize Client**:

```python
from openai import OpenAI
client = OpenAI(
    base_url='https://api.two.ai/v2',
    api_key=SUTRA_API_KEY
)
```

- **Test Basic Completion**:

```python
response = client.chat.completions.create(
    model='sutra-v2',
    messages=[{"role": "user", "content": "Tell me about AI advancements in 3 sentences."}],
    max_tokens=1024,
    temperature=0.7
)
print(response.choices[0].message.content)
```
