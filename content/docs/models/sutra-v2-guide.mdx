## ğŸ’¬ Step 4: Basic Completion

```python
response = client.chat.completions.create(
    model='sutra-v2',
    messages=[{"role": "user", "content": "Tell me about artificial intelligence in 3 sentences."}],
    max_tokens=1024,
    temperature=0.7
)
print(response.choices[0].message.content)
```

## ğŸ” Step 5: Streaming Response

```python
stream = client.chat.completions.create(
    model='sutra-v2',
    messages=[{"role": "user", "content": "Write a short story about a robot who discovers emotions."}],
    max_tokens=1024,
    temperature=0.7,
    stream=True
)

for chunk in stream:
    if chunk.choices and chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end='', flush=True)
```
